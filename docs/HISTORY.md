## Неделя 27 · 05–06 июля 2026 · Бот и автопубликация

Самый насыщенный период разработки — за двое суток выстроена вся инфраструктура.

**Дашборд:**
- Два деплоя из одного репо: `accelerator-weekly.vercel.app` (факт) и `accelerator-weekly-new.vercel.app` (WIP)
- `IS_NEW` — определение версии по hostname
- Нумерованные артефакты (разделитель ` · ` → список)
- OG-превью в стиле таблицы Менделеева (`og.png`): символ **Аи**, без дат

**Бот @reacto_robot:**
- Vercel Serverless Function (`api/webhook.js`) + Claude Haiku + GitHub Contents API
- Накопление данных через `appendStr` (дописывает через ` · `)
- Контекст диалога в `state.json`
- Групповой режим (отвечает на @mention и реплай)
- Команда `/id` — показывает chat_id и user_id

**Workflow (GitHub Actions):**
- Запуск каждый понедельник 06:00 МСК
- Архивирование → миграция активных → продвижение data-new → генерация шаблона
- Защита от двойной публикации (guard по номеру недели)

**Исправленные баги:**
- Бот писал в `data.json` вместо `data-new.json`
- Workflow обрывался из-за экранирования shell
- Активные пилоты без `done` оставались в активных при публикации

## Неделя 28 · 07–14 июля 2026 · Сергей и структура

К разработке подключился **Сергей Костюк** (@kostyuksergey) — первый внешний контрибьютор.

**Новые поля пилотов** (добавил Сергей):
- `owner` — ответственный от команды АИ (А / П / Н → Алексей / Полина / Надежда)
- `plan` — план на следующую неделю
- `risks` — риски / блокеры

**Новый раздел Бюджет** (добавил Сергей):
- Структура: TOTAL / CAPEX / OPEX / ФОТ по кварталам
- Нарастающим итогом — не сбрасывается при еженедельной публикации

**Бот — улучшения (Алексей):**
- Подтверждение перед сохранением: превью → «да / изменить»
- Режим правки `$set` — замена вместо дописывания
- Команда `/msg <chat_id> <текст>` — admin-only рассылка с превью
- Поддержка бюджета: глубокий мёрдж, системный промпт, превью
- Контекст: 20 сообщений (было 10), после сохранения остаётся 6 (было 0)
- `max_tokens`: 2048 (было 1024)

**Данные недели** (добавил Сергей через прямые коммиты):
- 4 инновации: WhoAI, GateForge, Veai, скаутинговая встреча
- `owner` / `plan` / `risks` заполнены для всех 28 пилотов
- PR-активность недели
